# Task 03 – Submission Form Layout

## Overview

This task was focused on building a **submission form page** based on a provided design.  
The goal was to recreate the layout using **HTML and CSS**, while correctly using the given images and styling the form elements to closely match the reference design.

The page includes a left informational section and a right form section for award submissions.

---

## Project Setup

This task is part of a larger repository and exists as a subfolder.

### To run this task locally:

1. Open the **main repository** on GitHub
2. Click **Code → Download ZIP**
3. Extract the ZIP file
4. Navigate to:
   Tasks/Task-03/
5. Open the main HTML file in a browser  
   _(Live Server can also be used)_

Make sure the **Images folder** remains in the correct relative path, as icons and logos are used in the layout.

---

## Features Implemented

- Complete submission form layout using HTML and CSS
- Two-column layout:
- Left section with heading, description, and deadline
- Right section with form inputs
- Form fields include:
- Text inputs
- Email input
- Checkboxes
- Textareas
- Submit button
- Custom styling for:
- Inputs and textareas
- Checkboxes
- Buttons
- Use of CSS Flexbox for layout structure
- Use of `clip-path` to match the angled background design
- External CSS file for better organization

---

## Image Usage

The design uses multiple images provided with the task, including:

- Logo image
- Favicon icon

All images are stored in the `Images` directory and are linked using **relative paths** to ensure the design works correctly after download.
