# Task 08 – JavaScript Practice Tasks

## Overview

This task contains six JavaScript practice exercises built with Tailwind CSS. Each task is a separate page, accessible from a central index.

---

## Project Setup

1. Navigate to: `Tasks/Task 08 - 16 February 2026/src/`
2. Open `index.html` in a browser (or use Live Server)

---

## Tasks Implemented

### Task 1: Responsive Carousel / Slider

- Image carousel with next/previous navigation buttons
- Auto-slide advances every 3 seconds
- Dot indicators for current slide position
- Responsive layout for different screen sizes

### Task 2: Calculator

- Basic operations: addition, subtraction, multiplication, division
- Clear (AC) and backspace functionality
- Clean grid-based button layout
- Error handling for invalid expressions

### Task 3: Digital Clock

- Displays real-time hours, minutes, and seconds
- 12-hour format with AM/PM indicator
- Shows current date (day, month, year)
- Updates every second

### Task 4: Temperature Converter

- Two-way conversion: Celsius to Fahrenheit and vice versa
- Live conversion as user types
- Clean, simple input-based interface

### Task 5: Random Password Generator

- Adjustable password length (6–40 characters)
- Options to include uppercase, lowercase, numbers, and special characters
- Copy-to-clipboard button with confirmation message
- Input field displays generated password

### Task 6: To-Do List App (Additional Task)

- Add, edit, delete, and mark tasks as completed
- Tasks saved to localStorage (persist after page refresh)
- Edit and delete buttons appear on hover
- Enter key support for quick task entry
