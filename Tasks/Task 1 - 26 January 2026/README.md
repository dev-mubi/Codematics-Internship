This task is part of the Web Development Internship program.

1. Install **VS Code**
2. Open the project folder in VS Code
3. Navigate to the task folder

---

## Features Implemented

- Used semantic HTML tags:
  - `header`
  - `nav`
  - `main`
  - `aside`
  - `section`
  - `footer`
- No use of `<div>` tags as per task instructions
- Layout structured to match the provided reference image
- Content centered horizontally and vertically in:
  - Main content section
  - Both side sections
- Basic CSS used only for layout, spacing, and alignment
- Clean and readable code structure
