# Task 02 – Landing Page Using Images

## Overview

This task focuses on building a **landing page layout** based on a provided design and image assets.  
The objective was to recreate the design using **HTML and CSS**, while properly using the given images for branding, sections, and visual layout.

The page represents a product-style landing page with a hero section, feature sections, and a footer.

---

## Project Setup

Since this task is part of a larger repository, it cannot be downloaded individually as a folder.

### To run this task locally:

1. Open the **main repository** on GitHub
2. Click **Code → Download ZIP**
3. Extract the ZIP file
4. Navigate to:
   Tasks/Task 02/
5. Open `index.html` (or the main HTML file) in a browser  
   _(Live Server can also be used)_

Make sure the **Images folder** remains in the correct relative path, as the layout depends on it.

All images provided for the task are stored in an `Images` directory and are used through **relative paths**.

Examples include:

- Logo image
- App screen previews

- Footer branding image

Proper image paths were maintained to ensure the design works correctly after download.
